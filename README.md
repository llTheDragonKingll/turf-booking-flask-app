# ⚽ Turf Booking Web App (Flask)

🚧 **This project is currently a work in progress.**

A backend-focused turf booking web application built using Flask and SQLite.
The app supports user signup/login, slot booking, and an admin view for checking bookings.

---

## 🚀 Features

* 👤 User signup and login
* 📅 Slot generation for upcoming days
* ✅ Turf slot booking
* 🗂️ Admin page to view booking data
* 💾 SQLite database for storing users and slots

---

## 🛠️ Tech Stack

* Python
* Flask
* SQLite
* HTML templates

## ⚙️ How to Run

1. Install dependencies:

```bash id="g92kda"

```

2. Run the app:

```bash id="h1t9c0"
python app.py
```

3. Open the local server link shown in the terminal.
